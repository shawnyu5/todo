# To Do list

A to do list written in go

## Features

-  [x] Add an task
   -  [x] Set task name
   -  [x] Set due date
   -  [x] Set description for task
   -  [x] Set priority level (lower number means more important)
   -  [x] Set reminder time (time relative to due date) - optional
-  [ ] Delete an item
-  [ ] Change due dates
